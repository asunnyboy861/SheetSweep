## Description
SheetSweep is the only iOS app that instantly cleans and standardizes messy spreadsheet files. Built for procurement teams, finance departments, and anyone who deals with supplier data.

STOP WASTING HOURS CLEANING SPREADSHEETS

Every month, professionals waste 2-4 hours per supplier file fixing inconsistent dates, duplicate rows, mixed currencies, and messy formatting. SheetSweep automates this entire process in seconds.

POWERFUL CLEANING FEATURES

• Duplicate Detection — Find exact and fuzzy duplicates even with typos using advanced JaroWinkler matching
• Date Standardization — Convert mixed date formats (MM/DD/YYYY, DD-MM-YYYY, etc.) into one consistent format
• Currency Unification — Detect and standardize mixed currency symbols ($, €, £, ¥) into a single format
• Extra Spaces Removal — Clean up trailing, leading, and extra whitespace
• Hidden Characters — Remove invisible Unicode characters that cause data errors
• Column Name Normalization — Standardize inconsistent column headers (e.g., "Price" → "Unit_Price")
• Missing Value Detection — Identify empty cells that should contain data

PRIVACY FIRST

All processing happens locally on your device. Your data never leaves your iPhone or iPad. No cloud uploads, no servers, no data collection. Works completely offline.

EASY WORKFLOW

1. Import — Select any .xlsx or .csv file
2. Scan — SheetSweep automatically detects all issues
3. Review — See every issue with suggested fixes
4. Fix — Apply all fixes with one tap
5. Export — Save clean data as XLSX or CSV

SUPPLIER TEMPLATE MEMORY

Save cleaning rules for repeat vendors. Next time you import from the same supplier, SheetSweep remembers your preferences and applies them automatically.

FREE TO START

Clean up to 3 files per month for free. Upgrade to Pro for unlimited cleaning, fuzzy duplicate detection, XLSX export, and supplier template memory.
